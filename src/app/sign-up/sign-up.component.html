The following displays the entropy (random numbers) used to generate the "seed phrase" above. The entropy below was generated using a random number generator, but you can edit the fields below to use your own randomness to generate the seed phrase (you can't go wrong with a set of dice). You can also enter your own seed phrase directly if you prefer.

Because there is a standard for generating seed phrases for Bitcoin (called BIP39), and because we use this same standard to generate BitClout seed phrases, BitClout seed phrases can also be used with any Bitcoin wallet that implements this standard under the m/44'/0'/0'/0/0 derivation path (e.g. Mycelium, Electrum). Similarly, any standard Bitcoin seed phrase you have can also be used as a BitClout seed phrase. Of course, all of this can be verified using a tool like the Ian Coleman mnemonic generator (must click "show entropy details") so you know this generator is doing what it says it is.
