The following displays the entropy (random numbers) used to generate the "forever password" (seed phrase) above. BitClout refers to seed phrases as "forever passwords" to make them more understandable and convey their permanence. The entropy below was generated using a random number generator, but you can edit the fields below to use your own randomness to generate the forever password (you can't go wrong with a set of dice). You can also enter your own forever password directly if you prefer.
Because there is a standard for generating seed phrases for Bitcoin (called BIP39), and because we use this same standard to generate BitClout forever passwords, BitClout forever passwords can also be used with any Bitcoin wallet that implements this standard under the m/44'/0'/0'/0/0 derivation path (e.g. Mycelium, Electrum). Similarly, any standard Bitcoin seed phrase you have can also be used as a BitClout forever password. Of course, all of this can be verified using a tool like the Ian Coleman mnemonic generator (must click "show entropy details") so you know this generator is doing what it says it is.
